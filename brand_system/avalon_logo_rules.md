# Avalon Escapes — Logo Rules

## Logo Type: Imagotype

The Avalon Escapes logo is an **imagotype** — the symbol and the wordmark are
designed to work together as one unit. Both elements are present in the primary logo.

---

## Logo Anatomy

```
[Butterfly+Compass symbol]  AVALON
                            - ESCAPES -
```

| Element | Description |
|---|---|
| **Symbol (Isotype)** | Butterfly whose body IS the compass needle — N at top, S at bottom. Wings are geometric, linear, illustrated. |
| **AVALON** | Amore Christmas OTF — large, elegant, ornamental serif. The emotional heart of the brand name. |
| **- ESCAPES -** | Montserrat Medium — smaller, all-caps, letter-spaced, flanked by em dashes. Subservient to AVALON in scale. |

---

## Logo Proportions

| Measurement | Value |
|---|---|
| Total width | 20 cm |
| Total height | 5 cm |
| Logotype (text) width | 10 cm |
| Text body height | 3 cm |
| Margin (each side) | 1 cm |

**Protection area (clear zone):** Maintain a minimum clear zone around the logo
equal to the size of the letter **"O"** in the main typeface (Amore Christmas).
Nothing — text, images, other graphics — should enter this zone.

## Correct Logo Usage

- Always use the provided logo files — never recreate from scratch
- Maintain the original proportions at all times
- Only use approved color versions (4 variants above)
- Maintain the required clear zone on all sides
- Place on backgrounds that provide sufficient contrast
- The symbol always sits to the **left** of the wordmark

---

## Incorrect Logo Usage — Never Do These

| Violation | Why it matters |
|---|---|
| Stretch or distort the logo | Destroys proportions and signals amateur handling |
| Alter the order of elements (e.g., text left, symbol right) | Breaks the visual grammar of the brand |
| Use off-palette colors | Undermines brand consistency |
| Use blue logo on blue background | Insufficient contrast — unreadable |
| Use green logo on blue background | Insufficient contrast |
| Change the proportion ratio between symbol and text | Disrupts the designed balance |
| Add effects (shadows, glows, outlines) | Cheapens the mark |
| Use the logo on a busy or clashing background | Renders it illegible |
| Recreate the logo in any other typeface | Breaks brand identity |

---

## Symbol-Only Use (Isotype)

The butterfly+compass can be used alone (without the "AVALON ESCAPES" wordmark)
in contexts where the brand is already established — e.g., as a social media profile
watermark, favicon, or decorative element in content.

Rule: the isotype alone should only appear where full brand context is already clear.
In new-audience content, always use the full imagotype.

---

## Logo in Social Media

- **Profile picture:** butterfly+compass isotype on navy background — simple, readable at small sizes
- **Content watermarks:** white isotype, low opacity (30–40%), bottom-right corner
- **Branded slides:** full imagotype (white on navy) for title/cover slides in carousels
- **Reels branding:** end card with full imagotype on navy; or subtle isotype in corner of frames
