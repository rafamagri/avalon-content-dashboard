# Avalon Escapes — Design System

## Color Palette

| Name | Hex | RGB | CMYK | Role |
|---|---|---|---|---|
| **Deep Navy** | `#0F2649` | 15, 38, 73 | 97/93/51/25 | Primary background, authority, depth |
| **Dark Teal Green** | `#213A3E` | 33, 58, 62 | 90/74/66/35 | Nature, ocean floor, secondary bg |
| **Medium Blue** | `#265196` | 38, 81, 150 | 93/76/13/0 | Dynamism, accents, highlights |
| **White** | `#FFFFFF` | 255, 255, 255 | — | Purity, contrast, text on dark |

### Color Usage Rules

**Backgrounds:** Use Navy `#0F2649` as the primary background.
Dark Green `#213A3E` works for secondary panels or nature-themed content.

**Text on dark:** Always white `#FFFFFF`. Never use off-palette colors for text.

**Accents:** Medium Blue `#265196` for buttons, links, highlights, and energy moments.

**On white/light:** Navy `#0F2649` for all primary text and logo usage.

**Never use:**
- Blue on blue, or green on blue (insufficient contrast)
- Off-palette colors not in the 4-color system
- Gradients that muddy the color purity

## Typography

### Primary Typefaces

| Font | Style | Use | Weight |
|---|---|---|---|
| **Amore Christmas** | Elegant ornamental serif | "AVALON" — the brand name | Regular |
| **Montserrat** | Modern geometric sans-serif | "ESCAPES", body text, UI, captions | Medium / Bold / Black |
| **Neue Frutiger World It** | Humanist italic | Supporting text (Adobe Fonts only — protected) | Italic |

### Type Hierarchy for Content

| Level | Font | Weight | Use |
|---|---|---|---|
| **Brand Name** | Amore Christmas | Regular | Logo wordmark "AVALON" only |
| **Headline / Slide Title** | Montserrat | Bold or Black | First slide of carousel, Reel text overlays |
| **Subheading** | Montserrat | Bold | Section headers, supporting hooks |
| **Body / Caption** | Montserrat | Medium | Slide body text, caption body, guide text |
| **Tagline / Decorative** | Montserrat | Medium (spaced) | "- ESCAPES -" treatment — letter-spaced caps |

### Typography Principles

- **Letter spacing on "ESCAPES":** The `-ESCAPES-` treatment uses tracked/spaced Montserrat all-caps
  with em dashes. This detail signals the brand's precision and restraint.
- **Hierarchy contrast:** Large serif (AVALON) + smaller spaced sans (ESCAPES) creates
  the tension between warmth and structure that defines the brand.
- **Never mix more than 2 weights in a single asset.** Restraint = luxury.
- **Line spacing:** Generous. Never cramped. Let the text breathe.

## Iconography

### Style: Linear, White, on Dark

All icons are line-only (no fills), white, used on dark backgrounds (navy or teal).
They are simple, clean, and elegant — not illustrated, not 3D, not colorful.

### Icon Library

| Icon | Context |
|---|---|
| Diving mask | Ocean content, Maldives, Sri Lanka, underwater adventures |
| Waves | Ocean, beach, coastal destinations |
| Seashell | Beachside luxury, island escapes |
| Fish / Marine life | Diving, snorkeling, ocean wildlife |
| Sun | Tropical destinations, resort content, summer escapes |
| Binoculars | Discovery, exploration, nature-based travel |
| Location pin | Destination reveal, "where we went" content |
| Palm tree | Tropical, Maldives, Brazil, Caribbean-adjacent |
| Passport | Travel planning, new journey, departure content |

### Icon Usage Rules

- Use icons as **decorative accents** on dark backgrounds — not as primary visual elements
- Always white line icons on `#0F2649` or `#213A3E`
- Do not use filled icons, colored icons, or icons on white/light backgrounds
- In carousels: subtle icon use as slide dividers or destination markers only

## Visual Direction

### Photography Style

| Attribute | Avalon direction |
|---|---|
| **Light** | Golden hour, soft natural light, no harsh flash |
| **Palette** | Deep blues, ocean teals, warm terracotta (Colombia/Türkiye), lush greens |
| **Composition** | Centered calm, breathing room, not cluttered |
| **People** | Present but not posed — real moments, backs to camera, in the water |
| **Hotels/Spaces** | Intimate, boutique, private — never the lobby crowd shot |
| **Ocean content** | Underwater, from water, boats, docks — immersive not postcard |

### What to avoid visually:
- Stock photo energy (too perfect, no soul)
- Busy compositions with text everywhere
- Overly saturated or HDR-processed images
- Wide-angle crowd shots of popular landmarks

## Brand in Social Media Context

### Instagram grid visual language:
- Deep navy/teal tones dominate
- White text overlays on dark imagery
- Montserrat Bold/Black for any text slides
- Icons used sparingly as elegant punctuation
- Butterfly symbol can appear as a watermark on brand-owned content

### Contrast rule for content:
Every branded post should pass the test: "Does this look like it costs money?"
If yes — it's on-brand. If it looks like it could be from any travel account — reframe it.
