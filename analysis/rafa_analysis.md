## 5. Caption style

- **Length:** Medium-short for the best performers. Structured as short stanzas, not paragraphs. White space is used deliberately.
- **Voice:** First-person, deeply personal, confessional. Reads like a personal essay or a letter.
- **Structure pattern:** Hook → Emotional escalation (often 3 short parallel lines) → Insight/resolution → Universal closing line
- **Parallel structure:** Rafa uses triplets repeatedly: "At first, it feels scary. / Then it feels quiet. / Then it feels like freedom."
- **Language:** English. Occasional Portuguese words used intentionally (saudade, brasil) to assert cultural identity.
- **Tone:** Philosophical, warm, vulnerable, inspiring. Never salesy. Never generic.
