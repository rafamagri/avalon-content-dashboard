# Viral Influencer Trend Reference Group — Analysis

**Conceptual name:** `viral_influencer_trend_reference_group`
**Physical folder:** `viral_reference_group/` (kept for stability)

10 accounts treated as one collective benchmark pool.

## Purpose

Study viral influencer content, trend formats, and content structures that Avalon Escapes
can adapt or recreate. Social media is heavily trend-based — the goal is not to avoid
replication entirely. The goal is to understand what can be replicated safely and strategically.

These accounts are NOT competitors. They are a combined viral/trend reference pool.

## Three-tier classification

Every pattern, hook, script formula, and caption structure in the analysis is tagged with one of:

| Tier | Name | Meaning |
|---|---|---|
| 1 | **Direct Trend Adaptation** | Format is already trend-based. Recreate with Avalon footage, voice, destinations. (POV, "send this to", list Reels, keyword CTAs, invitation formats, transition trends) |
| 2 | **Structural Replication** | Copy the structure — hook formula, emotional arc, pacing, caption shape, CTA style — but not the exact words, footage, or personal story. |
| 3 | **Inspiration Only** | Do not copy. Too personal, too creator-specific, or too close to their brand identity. Use only for directional ideas. (Personal manifestos, signature phrases, private experiences, unique editing identity) |

## Output file

`viral_pattern_library.md` — the full A–G report

Sections:
- **A. Viral Pattern Library** — 12 patterns, each tagged with classification tier
- **B. Hook Bank** — 20 reusable hook formulas, each tagged with tier
- **C. Reel Script Formulas** — 5 script structures, each tagged with tier
- **D. Caption Formulas** — 8 caption structures, each tagged with tier
- **E. Visual and Editing Patterns** — POV formats, transitions, text-on-screen, opening shots
- **F. Avalon Adaptation Guide** — top 5 patterns mapped to Avalon's destinations and voice
- **G. Content Scoring System** — rubric to rate any Avalon post idea before creating it (out of 100)
